## ReconPay
O ReconPay é uma plataforma backend para conciliação financeira, criada para comparar transações internas de uma empresa com arquivos ou registros externos de liquidação financeira, identificando divergências como valores diferentes, taxas incorretas, pagamentos ausentes, duplicidades, chargebacks e diferenças de data de liquidação.

O projeto simula um problema real enfrentado por empresas de pagamento, fintechs e bancos, como Cielo, Stone, Itaú, Santander.
